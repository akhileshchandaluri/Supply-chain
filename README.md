## 🚀 Quick Start

### 1. Start the FastAPI Intelligence Hub
```bash
cd backend
python -m venv venv
venv\Scripts\activate      # On Windows
pip install -r requirements.txt
python main.py
```

### 2. Start the React Dashboard
```bash
cd frontend
npm install
npm run dev
```
